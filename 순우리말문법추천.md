## 💡 순우리말 문법 예제

### 기본 문법
```
그릇 나이를 스물다섯으로 뜻정하기.
그릇 이름을 "김철수"로 뜻정하기.

혹시 나이가 스무보다 크거든
    "어른입니다"를 내보이기.
그렇지않으면
    "아이입니다"를 내보이기.
마침.
```

### 함수 정의
```
일꾼 인사하기를 이름으로 뜻정하기
    "안녕하세요"를 내보이기.
    이름을 내보이기.
    "님!"을 내보이기.
마침.

인사하기를 "세종대왕"으로 부르기.
```

### 반복문
```
그릇 횟수를 다섯으로 뜻정하기.
그릇 현재를 하나로 뜻정하기.

현재가 횟수보다 작거나같은동안 되풀이하기
    "반복"을 내보이기.
    현재를 내보이기.
    그릇 현재를 현재와 하나를 보탠값으로 뜻정하기.
마침.
```

### 계산
```
그릇 첫수를 열으로 뜻정하기.
그릇 둘째수를 스무로 뜻정하기.
그릇 합을 첫수와 둘째수를 보탠값으로 뜻정하기.

"계산 결과: "를 내보이기.
합을 내보이기.
```

## 🎨 완전한 순우리말 예제

```
// 세종대왕의 순우리말 프로그래밍

그릇 백성수를 천만으로 뜻정하기.
그릇 글아는백성을 하나로 뜻정하기.

"한글 창제 전 상황:"을 내보이기.
"전체 백성: "을 내보이기.
백성수를 내보이기.
"글 아는 백성: "을 내보이기.
글아는백성을 내보이기.

일꾼 한글창제하기를 뜻정하기
    "훈민정음을 창제합니다!"를 내보이기.
    그릇 글아는백성을 백성수와 같은값으로 뜻정하기.
    "이제 모든 백성이 글을 읽을 수 있습니다"를 내보이기.
마침.

한글창제하기를 부르기.

"한글 창제 후:"을 내보이기.
"글 아는 백성: "을 내보이기.
글아는백성을 내보이기.

혹시 글아는백성이 백성수와 같거든
    "대성공입니다!"를 내보이기.
마침.
```
